# Implementation Plan: Hidden Notes

**Branch**: `010-hidden-notes` | **Date**: 2026-06-20 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/010-hidden-notes/spec.md`

## Summary

Add inline hidden notes to the melody composer and viewer. Text wrapped in asterisks (`*text*`) is displayed in a muted color. In the editor, asterisks are always visible. In the view page, asterisks are never shown — only the muted inner text. Hidden notes are purely visual — note parsing and transposition operate normally on all tokens including those inside hidden notes.
